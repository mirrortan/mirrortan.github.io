## Front Matter

以下是在个人博客中使用到的 yaml 变量：

- layout: 布局，依据 `layouts` 文件夹下对应文件来组织内容；
- title: 文章标题；
- categories: 分类；
- description: 文章的描述，在分享的时候使用；
- tages: 标签；
- next: 下一篇有逻辑关联的文章
- previous: 上一篇有逻辑关联的文章
